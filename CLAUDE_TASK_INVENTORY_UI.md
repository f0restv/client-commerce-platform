# Task: Inventory Management UI

## Objective
Build inventory dashboard for users to manage their items.

## Steps
1. Create /dashboard page - overview of user's inventory
2. Create /inventory page - list view with filters (coin type, grade, status)
3. Item detail page /inventory/[id] - edit, view history, pricing
4. Bulk actions: select multiple, update status, delete
5. Status workflow: draft → listed → sold → paid
6. Connect to existing Prisma Inventory model
7. Add search/filter by coin type, date, grade
